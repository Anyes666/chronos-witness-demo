# 时序证人 · Playtest 包

## 测试目的

验证「证词漂移」核心玩法是否成立。具体要回答：

> 玩家是否能从多轮证词变化中，识别出稳定事实，并以此为基础完成推理？

## 测试链接

https://<你的GitHub用户名>.github.io/chronos-witness-demo/

## 推荐设备

| 设备 | 体验 |
|------|------|
| 桌面 Chrome / Edge | 3D 档案馆模式（推荐） |
| 手机 Chrome / 鸿蒙浏览器 / Safari | 2D 调查模式（自动适配） |

## 测试时长

20-35 分钟

## 玩家须知

- 这是早期 Demo，重点是玩法验证，不是画面完成度。
- 回溯会导致证词和环境发生变化。
- **变化不是 bug**——变化本身是线索。
- 请尽量说出你最终指控的依据。
- 通关后请导出日志（点击结局页的「导出 Playtest 日志」按钮），并发回给测试组织者。

## 测后问卷

通关后请回答以下问题：

1. **你什么时候第一次意识到证词发生了变化？**
2. **你觉得证词的变化是线索还是干扰？** 为什么？
3. **你是否找到了某些「一直没变」的信息？** 请举例。
4. **你的最终指控是推理出来的，还是猜测的？** 依据是什么？
5. **证词对照板有没有帮助你？** 你用了几次？
6. **哪个地方最让你困惑？**
7. **有没有哪个瞬间让你觉得「啊，我懂了！」？** 是什么？
8. **你是否愿意继续玩第二个案件？**
9. **（桌面端）3D 操作是否顺畅？** WASD 移动、鼠标转向、E 键交互是否好用？
10. **（手机端）2D 模式是否能顺利通关？** 有没有操作困难？

## 核心验证假设（来自设计文档）

| 假设 | 成功指标 | 失败指标 |
|------|---------|---------|
| H1 玩家理解「回溯是污染非重开」 | ≥70% 回溯前犹豫 ≥3 秒 | ≥40% 5 秒内连点回溯 |
| H2 玩家能区分漂移与稳定事实 | ≥60% 指控理由引用 ≥1 稳定事实 | ≤30% 意识到存在稳定事实 |
| H3 玩家自然形成「比对漂移」习惯 | ≥50% 二次回溯后 60 秒内主动开对照板 | ≤20% 主动开对照板 |
| H6 乐趣高于挫败 | 乐趣均分 ≥3.5/5 | 挫败 > 乐趣 |
